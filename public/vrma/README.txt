VRM Animation files
===================

This folder is served by the local app at /vrma/.

Included:
- pixiv-test.vrma — official sample clip from pixiv/three-vrm.

VRoid/BOOTH official sample pack:
- Source: https://vroid.booth.pm/items/5512385
- Downloaded archive: VRMA_MotionPack.zip
- SHA-256: 64d6e87d12ad0e43daaf4f261f74b05322329b9f18b9b7ab4da6f9611b995af8
- The VRM Editor has preset buttons for the seven official motions named by VRoid:
  Show full body, Greeting, Peace sign, Shoot, Spin, Model pose, Squat.

The BOOTH motion binaries are installed on this machine for local production use and are
ignored by Git so they are not redistributed from the public repository.

Those preset buttons play these files:
- public/vrma/vroid/show-full-body.vrma
- public/vrma/vroid/greeting.vrma
- public/vrma/vroid/peace-sign.vrma
- public/vrma/vroid/shoot.vrma
- public/vrma/vroid/spin.vrma
- public/vrma/vroid/model-pose.vrma
- public/vrma/vroid/squat.vrma

Original BOOTH readmes are preserved in public/vrma/vroid/.

You can also use the VRM Editor floating panel's Import .VRMA button to play any
downloaded .vrma file immediately, regardless of filename.
